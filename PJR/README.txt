Afin de pouvoir lancer le dispositif de jeu, il faudra :

1) Télécharger et installer l'environnement node.js à l'adresse suivante : https://nodejs.org/en/download/
2) A la première execution, assurez vous d'avoir une connexion internet afin de pouvoir télécharger les ressources nécessaires
3) Executer start.bat à chaque fois que vous souhaitez lancer le server
4) FACULTATIF : Si votre navigateur web ne se lance pas de lui même, veuillez l'ouvrir à l'adresse suivante : http://localhost:5000

Attention, le jeu nécessite un navigateur récent.



Si vous souhaitez utiliser votre appareil en tant que serveur web afin de permettre à d'autres appareils de jouer sans qu'ils aient à installer quoi que ce soit :
1) Assurez-vous que l'appareil se connectant soit sur le même réseau que votre machine servant de serveur web
2) Naviguez (via un navigateur internet de l'appareil se connectant) à l'adresse actuelle de votre serveur web suivi de son port,
	ex : 192.168.1.15:5000
    Pour récupérer l'adresse ip de votre machine, executez via une invite de commande : 'ipconfig' (sur windows) ou 'ip a' (sur linux ou mac)
    Le port est par défaut 5000
3) Votre appareil devrait désormais avoir accès au jeu
